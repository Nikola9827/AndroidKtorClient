# AndroidKtorClient

A Jetpack Compose Android application that communicates with a Kotlin Ktor backend. Demonstrates full-stack interaction, including asynchronous network requests using coroutines and MVVM architecture.

**Tech stack:** Kotlin, Jetpack Compose, Ktor Client, MVVM, Coroutines

**Features:**
- Displays messages from the backend.
- Updates UI based on user interactions.
- Uses ViewModel and Repository for state management.
